# 🧾 Control Feria

Aplicación web para control de stock y ventas diarias.

## 🚀 Funcionalidades

- Gestión de productos (nombre, código, precio, stock)
- Registro de ventas
- Ingreso de mercadería
- Búsqueda rápida por nombre o código
- Autocompletado en ventas
- Historial por fecha
- Exportación a CSV
- Envío de resumen por WhatsApp
- Cálculo automático del total vendido
- Cálculo del valor total en stock

## 🛠️ Tecnologías

- HTML
- CSS (Bootstrap 5)
- JavaScript
- LocalStorage

## 📱 Uso

1. Agregar productos
2. Registrar ventas
3. Consultar historial
4. Exportar o enviar resumen
